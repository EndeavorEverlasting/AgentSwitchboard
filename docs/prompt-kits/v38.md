# AI Harness Prompt Kit V38

AgentSwitchboard vendors one offline, integrity-checked snapshot of the AI Harness Prompt Kit so agents and operators can select and render the right prompt without browsing the workbook manually.

Web Excel Repair Triage owns extraction and canonical content. AgentSwitchboard owns offline selection, exact variable substitution, execution-surface enforcement, and consumer tests. Updating the snapshot requires a reviewed source commit and updated hash evidence.

## Operator entrypoint

```powershell
.\Select-AgentSwitchboardPrompt.cmd -Mode List
```

Search the kit:

```powershell
.\Select-AgentSwitchboardPrompt.cmd `
  -Mode Search `
  -Query "pull request repair" `
  -ExecutionSurface regular_ai_prompt
```

Inspect one prompt:

```powershell
.\Select-AgentSwitchboardPrompt.cmd `
  -Mode Show `
  -PromptId P14 `
  -ExecutionSurface regular_ai_prompt `
  -AsJson
```

Render a regular AI prompt:

```powershell
.\Select-AgentSwitchboardPrompt.cmd `
  -Mode Render `
  -PromptId P07 `
  -ExecutionSurface regular_ai_prompt `
  -Variable `
    "xyz_repo_or_path=EndeavorEverlasting/AgentSwitchboard" `
    "xyz_sprint_name=prompt registry consumer" `
    "xyz_owned_scope=prompt selector, skill, docs, tests" `
    "xyz_forbidden_scope=provider calls and unrelated GNHF runtime changes" `
    "xyz_plan_directory=docs/prompt-kits/v38.md" `
  -OutputPath "$env:TEMP\agentswitchboard-p07.txt"
```

Render a GNHF launch artifact only through the GNHF surface:

```powershell
.\Select-AgentSwitchboardPrompt.cmd `
  -Mode Show `
  -PromptId P38 `
  -ExecutionSurface gnhf_launch_artifact
```

The selector rejects a regular AI prompt requested as a GNHF artifact and vice versa.

## Machine behavior

The selector:

- verifies the vendored registry SHA-256 from `source.json`;
- decodes the local gzip/base64 JSON bundle in memory and verifies all 45 prompt-text hashes on every invocation;
- supports `List`, deterministic substring `Search`, `Show`, and exact-variable `Render`;
- rejects undefined or duplicate variables;
- requires every prompt variable unless `-AllowUnresolved` is explicit;
- refuses to write rendered output beneath `.ai/prompt-kits/v38/`;
- makes no network, provider, model, Drive, GitHub, or target-runtime call.

## Proof ceiling

This integration proves a pinned offline snapshot, integrity validation, deterministic selection, exact substitution, and execution-surface rejection. It does not prove that a chosen prompt is the best judgment for every situation, that a rendered GNHF artifact was executed, that a provider is available, or that repository work succeeded.
