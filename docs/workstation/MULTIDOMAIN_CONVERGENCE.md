# Multi-Domain Workstation Convergence

## Resolution policy

```text
canonical agent command
-> healthy native executable outside the managed wrapper directory
-> explicit Windows bridge only when allowed
-> action required
```

The managed destination is `~/.local/agent-switchboard/bin`. Every supported
agent publishes canonical, `_native`, and `_win` commands. OpenCode bridges via
`cmd.exe` to its `.cmd` entrypoint; AGY and Goose use their `.exe` interop
entrypoints. Bash aliases are diagnostic observations, not executable contracts.

## Proof ceiling

Contract, fixture, command-resolution, and shell-loop proof do not prove provider
responses or authentication. Live install and interactive agent behavior require
separate runtime evidence. No macOS implementation is claimed.
