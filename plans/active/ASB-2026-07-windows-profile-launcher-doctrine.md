# Windows Profile Contract

AgentSwitchboard is the canonical owner for platform profile launch contracts.

The Windows Profile uses WezTerm and the operation name `open-or-activate`. SysAdminSuite is the consumer and certifier. Linux and Android are separate profiles, and Android settings may differ.

This delivery installs policy and validation only. Later reviewed changes must provide the AgentSwitchboard implementation and the SysAdminSuite adoption evidence.

No installed launcher, terminal behavior, device configuration, or consumer certification is proven here.
